Vue2.5开发去哪儿网App
技术栈和主要框架

📦 Vue 全家桶：vue2.5 + vuex + vue-router + webpack

📌 ES6

📡 网络请求：axios

未接入后台简单接口的请参考其他分支，例如 origin/city-components

项目运行

# 克隆到本地
git clone 项目地址

# 安装依赖
npm install

# 开启本地服务器localhost:8088
npm run dev

# 发布环境
npm run build
